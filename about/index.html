# Cuterwrite 的技术博客

👀欢迎来到 Cuterwrite 的技术博客！

# ✋ 关于我

我是 Cuterwrite，一名在读博士生，博客专注于以下领域：

  • 高性能计算：这是我的主业，主要研究并行计算、分布式系统和高性能计算架构。我致力于在大规模集群（超算）上进行应用优化，以提升系统的计算效率和性能。
  • 操作系统：主要是 Linux 系统的内核机制和性能优化策略，一些底层细节对于我来说是挺有帮助的。
  • 全栈开发：对前端和后端的技术都有涉猎，主要是为了自己的博客和一些 Toy 项目，也会尝试一些新技术，像是前端的 Vue、React，后端的 Golang、Java、Rust 等。
  • 人工智能：虽然这不是我的专业研究方向，以前只是接触和上手过，但随着近年来大模型的快速发展，我也对人工智能领域产生了浓厚兴趣。我正在了解如深度学习、自然语言处理等 AI 技术，并基于开源代码和模型进行一些探索实践。

我个人目前希望把 idea 落实下来并发出论文，尽快完成自己的博士学业。未来打算找家在大湾区的公司，最好是高性能计算领域相关的，先搞钱养活自己，然后再去做一些自己想做的事情。至于博客我会在业余时间不定期保持更新的。

希望通过不断地学习，提升自己的技术水平。如果你对我的博客感兴趣，欢迎关注我的动态，也欢迎在评论区留言，我会尽快回复的。

# ⌛️ 关于网站

Cuterwrite’s Blog 搭建于 2021 年，刚开始只是本科期间的一些技术笔记和学习心得，后来逐渐发展成了一个技术博客。网站采用 Hugo，托管于 Github Pages，并绑定域名 cuterwrite.top。

  • 评论系统：使用的是 Twikoo ，数据托管在免费的 MongoDB Atlas 上。
  • 网站统计分析：使用的是 Umami ，数据托管在免费的 Neon 的 PostgreSQL 服务上。
  • 搜索功能：使用的是自建的 MeiliSearch 服务。
  • 至于左边的画板和画册：分别基于开源项目 hugo-theme-gallery 和 excalidraw-cn 搭建的。
  • 网站 CDN：使用的是 语鹿云盾 ，稳定性和速度都很不错，基础套餐对于我的站点来说就已经足够了。要知道没加 CDN 之前，Github Pages 在国内的访问速度是真的慢，有时候还会被墙，打不开网站，加了 CDN 之后速度明显提升了不少。

Cuterwrite 的命名没有特别的含义，只是博主当初玩游戏的时候实在不知道取什么 ID 好，就在某个 ID 生成网站上随机生成了一大堆然后挑了这个，也挺好记的，所以后面就一直沿用至今。

Cuterwrite’s Blog 也一直在努力提升用户体验。因此，我时不时就对版面进行优化，添加新功能，力求让每一位访问者都能有良好的阅读体验。
