@startuml
!theme mars
actor Application 
participant "io_uring" as io_uring 
participant "I/O Subsystem" as io_subsystem 
participant "Submission Queue" as sq 
participant "Completion Queue" as cq 

Application -> io_uring: 提交I/O请求
io_uring -> sq: 将请求添加到SQ 
sq -> io_subsystem: 将请求添加到I/O队列

loop 等待I/O完成
io_subsystem -> cq:I/O完成事件 
cq -> io_uring: 通知I/O完成
io_uring -> Application: 通知I/O完成 
end

Application -> io_uring: 获取已完成的I/O结果
io_uring -> cq: 获取CQE cq -> io_subsystem: 获取I/O结果 
io_subsystem -> sq: 获取SQE sq -> io_uring: 获取I/O请求
@enduml
